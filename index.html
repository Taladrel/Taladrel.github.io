Project Idea

The basic principle of my project idea is to form a phone application and website which works in unison with each other that is capable of connecting farmers with potential seasonal workers and overseas working visa holders that need to complete their hours for their second and third year working visas. This is in hopes of creating a safe and simple platform for workers and farmers to be able to work together with each other.

The motivation for creating this service comes from meeting my current partner Charlotte. When we first met she was on a working visa and only planning to stay for one year. When we looked at getting the hours she needed to gain her second-year visa we also started to realize how hard it was to find local work for her, not only that but it can also be quite dangerous as exploitation and abuse are not unheard of when it comes to backpackers.

The features I would like to implement in the application are:
  • A profile system that for farms will include the name of the farm, the location, a simple job description, if tools or any previous experience are required and if accommodation is available on site. There will also be a feature that can recommend the nearest accommodation available to the workplace. For workers, the profile will include Their name, previous workplaces if any and the location of they are looking for work in. Each profile type will be able to contact each other to say they are interested.

  • A rating system will also be implemented for all profiles so farmers and workers are able to rate each other so both parties are able to quickly tell and easily tell if something is wrong. This would be a star system quickly visible to the user on the front of the profile they are looking at. With further information about each review available after opening the profile. This feature will help workers tell if they are going to a safe environment and that others have been paid properly and will help farmers know if the person they are going to potentially employ is a responsible and hard worker.

  • Part of the working visa conditions are you must work a set number of days to be able to gain your next visa, the current being 3 months to gain your second year and then another 6 months to gain your third. You must be able to prove this and that’s where this feature would come in. A farmer would be able to add days to your profile once you’ve finished your term with them. This would make it easy to prove on your application from a simple and easily readable report.

  • Implemented in the application will be a payroll system this feature will not be mandatory to use but only there to make the lives of both parties easier a worker can input the hours or pieces they have done depending on the agreement and the farmer will be easily able to pay it into the workers designated bank account they have added to the service.

The software required would be a compiler that is capable of compiling a back end language possibly like PHP or python as well as java and swift for phone development, A software such as Atom to be able to write HTML for the website although a service like WordPress might also be applicable. Hardware requirements would either be the purchase or hire of servers to be able to host the information required and a variety of different phones to be able to test functionality with.

The skills required to create this project would be either to learn or hire someone capable of coding the specific coding languages required. I think learning all the required languages required is possibly an unreasonable goal as it would take too long to learn each language but also the development time would be considerably longer. I also think that the server costs may also be an issue so a way to make money from the service may be needed which may make users unwilling to use it.

The end outcome of this project would be to reduce the difficulty of finding seasonal work for people and making farmers' lives easier by reducing the time and money needed to spend looking for workers. While also increasing the safety of both parties involved. The impact this service could have could be quite large as it could scale easily to other industries for example construction when unskilled or limited skilled workers are required.
